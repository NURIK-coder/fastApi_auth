
Создать бота для просмотра событий и участия на них.

Создать таблицу для хранения событий
id (идентификатор события),
event_name (название события),
event_date (дата и время события),
location (место проведения события),
description (короткое описание события),
created_at (дата и время создания события).


Создать таблицу User
id,
tg_id
role
name
contact (optional)
events (M2M)


Для админа:
Просмотр событий
Просмотр одного события (по его номеру)
Изменение события
Удаления события
Напоминание всем юзерам какого-то события, что они должны прийти на какое-то событие. (При выборе какого-то события будет кнопка - напомнить всем. При нажатии на нее, через api получаем всех юзеров, которые относятся к этому событию. И всем этим юзером отправляется сообщение в боте).


Для юзера:
Просмотр всех событий
Просмотр своих событий
Принять участие
Отказаться от участия в событии


API запросы:

get_user_by_tgid
register

create_event
update_event
delete_event
get_event_by_id/
get_users_by_event_id/


participate
unparticipate
